Помощь

Программное обеспечение - Конвертер данных Galileos в DICOM

Что это за сервис?

Наш сервис позволяет вам легко и быстро конвертировать данные с томографа Galileos в стандартный формат DICOM. Это делает их доступными для широкого круга программного обеспечения, используемого для визуализации, анализа и обработки медицинских изображений.

Как использовать сервис?

Процесс конвертации прост:

  1. Загрузите архив с файлами данных Galileos на страницу сервиса. (время загрузки зависит от размера архива)
  2. Дождитесь окончания конвертации
  3. Загрузите полученный DICOM файл из личного кабинета или по ссылке из письма направленного на указанную при регистрации почту

Какие типы файлов Galileos поддерживаются?

Файлы Galileos, как правило, не имеют формата

Есть ли ограничения по размеру файлов?

Максимальный размер загружаемого файла 700 MB.
